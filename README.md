# EduAI

**EduAI** is an AI-powered educational website that helps students learn faster and smarter. It allows users to summarize educational text, extract main points, and generate AI-style quiz questions that can be answered and checked instantly.

- **Text Summarization:** Paste any educational text, and EduAI will generate a concise summary.  
- **Main Points Extraction:** Highlights key points from the text to make learning easier.  
- **AI-Generated Questions:** Creates questions based on the pasted text.  
- **Answer Checking:** Students can input their answers and immediately see which ones are correct.  
- **Dark Mode:** Toggle between light and dark themes for comfortable viewing.  
- **Language Toggle:** Switch between English and Arabic for multi-language support.  
- **Sidebar Navigation:** Easily navigate through input, summary, main points, quiz, and results sections.  
- **Custom Logo:** School or institution branding is supported.
